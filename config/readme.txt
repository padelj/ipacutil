This directory contains some sample config files:

ipac2.cfg -- sample for I-PAC2 with 28 inputs.  These are the default settings
ipac2_32.cfg -- sample for I-PAC2 with 32 inputs.  These are the default settings
ipac4.cfg -- sample for I-PAC4.  These are the default settings
ipacve.cfg -- sample for I-PAC VE.  These are the default settings
minipacve.cfg -- sample for Mini-PAC VE.  These are the default settings
ipac2_macro.cfg -- sample for I-PAC2.  This shows how to use a macro

ipacve_cred+vol.cfg -- sample for I-PACVE submitted by Michael S.G.
ipacve_default.cfg -- sample for I-PACVE submitted by Michael S.G.
ipacve_full-lock.cfg -- sample for I-PACVE submitted by Michael S.G.
ipacve_lock+vol.cfg -- sample for I-PACVE submitted by Michael S.G.

Note: the I-PACVE files should also work with the I-PAC2 32-input boards.

Board Types:
------------
0 - I-PAC2 (28 input)
1 - I-PAC2 (32 input)
2 - I-PAC4 (56 input)
3 - I-PACVE (56 input)
4 - MINI-PACVE (36 input)
